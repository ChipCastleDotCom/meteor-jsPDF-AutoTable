meteor-jspdf-autotable

Meteor package for AutoTable plugin to jsPDF.

BACKGROUND

This repo is simply a wrapper to make the AutoTable plugin available on Atmosphere as a Meteor package.

INSTALLATION

Change into your meteor project's directory and install the package:

meteor add chipcastledotcom:jspdf-autotable

USAGE

For usage, please AutoTable plugin to jsPDF visit the documentation for the plugin.
